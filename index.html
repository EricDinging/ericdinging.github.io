Selected Projects

Propius is a federated learning (FL) resource management system, employing a microservice architecture based on gRPC protocol and Redis database. The system provides a clean abstraction on top of the device-level heterogeneity over the network, and supports various scheduling policies for different objectives. Venn, an advanced scheduling policy for FL job completion time (JCT) optimization, is deployed in Propius. Evaluation on GPU clusters showcased that the system could improve average JCT by 88% compared to random allocation.

Tennis Ball Collector is an autonomous robot dedicated to collecting objects such as tennis balls. It can:

  • Search objects through a pixycam running OpenCV models, and a direction control feedback loop
  • Collect objects using a spinning rotor
  • Move around freely, driven by two powerful motors connected through H-bridges
  • Detect obstacles using ultrasonic sensors with a moving average algorithm for noise filtering
  • Do all the stuff by itself thanks to a well-designed finite state machine

WASP: Wearable Analytical Skin Probe, is a wearable closed-chamber hygrometer-based device. WASP uses a high-performance communication protocol that operates atop I2C and Bluetooth Low Energy protocols. This protocol enables low-latency and reliable communication between microcontrollers, ensuring high-fidelity data collection. WASP has been successfully deployed in experimental setups for measuring insensible sweating (TEWL) and tracking skin dehydration-rehydration cycles.

Light Strider: Embark on an exciting journey! Light Strider is a captivating front-end web game written in ELM. You'll take charge of directing light beams through intricate mazes, strategically employing mirrors and light splitters to avoid obstacles. The game leverages intricate graph algorithms to generate infinite game maps. Give it a try!
